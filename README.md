# executado em desenvolvimento

1. Clonar o repositorio
2. Executar

``` 
 npm install
```
3. Ter Nest Cli instalado
```
npm i -g @nestjs/cli
```

4. Levantar a base de dados
```
docker-compose up
```

## Stack usdado
* MongoDB
* Nest
